CKS Course Content

This module covers the basics of Kubernetes security, including the security challenges associated with Kubernetes and how to mitigate common risks. Topics include securing Kubernetes clusters, managing vulnerabilities, and securing container runtimes.

In this module, you will learn how to securely set up a Kubernetes cluster. Topics include securing the API server, controlling access to the cluster, and configuring proper RBAC policies to limit the potential attack surface.

This section covers pod security. You'll learn how to secure your pods by configuring security contexts, using security policies, and limiting the privileges of pods to minimize security risks.

Network security is crucial for securing Kubernetes applications. This module covers best practices for securing network communication in Kubernetes, including using Network Policies, securing communication between pods, and securing ingress and egress traffic.

Container security focuses on securing the container runtime. This module will help you learn how to secure the containers themselves by scanning images for vulnerabilities, implementing runtime security policies, and securing container registries.

Learn how to effectively monitor, log, and respond to security incidents within your Kubernetes clusters. This module covers configuring logging and monitoring, setting up alerts, and responding to incidents.

This module teaches you about Kubernetes auditing and compliance practices. You'll learn about auditing Kubernetes activities, generating audit logs, and using tools to ensure compliance with security standards like CIS and NIST.
